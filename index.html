About the Project

Our project aims to create an application for the Northrop Grumman Corporation (with guidance from Phil Barnes of NGC) that utilizes Blender and Python to generate three-dimensional aircraft models. For the aircraft geometries, we utilize parametric cubic splines, where functions for x, y, and z coordinates are parameterized with an independent variable, in our case, “time.” Users will be able to input a set of points, and the Python code will generate functions to visualize curves around the points. They will also be able to edit the aircraft components and readjust specifications to their liking. Our end goal is to be able to generate models capable of being exported for 3-D printing.
